void bignum_assoc ( bignum_t *  num,
bignum_elem_t *  arr,
const size_t  num_elements 
)

Associate num_elements in arr with num.

After calling bignum_assoc() the elements arr[0] to arr[num_elements-1] will be treated as a big number, which can be accessed through num.

This code will create an array big enough to hold a 1024 bit number and associate the variable x with it:

bignum_t x;
bignum_elem_t x_elements[BIGNUM_1024];
bignum_assoc(&x, x_elements, BIGNUM_1024);
Parameters
numThe number with which the elements in arr will be associated with.
arrThe array holding the actual elements.
num_elementsNumber of elements from arr to associate with num.
